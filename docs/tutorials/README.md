# Tutorials

Tutorials are the guided path through KohakuTerrarium.

Use them when you want to learn by building something concrete step by step.

Guides answer focused task questions. Reference pages give exact lookup information. Concepts explain the architecture. Tutorials are the bridge that turns those things into an actual learning path.

## Why tutorials are separate from guides

KohakuTerrarium has multiple ways to build systems:

- config-driven creature authoring
- terrarium composition
- extension through custom modules and plugins
- programmatic composition in Python

A guide usually answers "how do I do X?"
A tutorial answers "show me how this style of building actually feels end to end."
